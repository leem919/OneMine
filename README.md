# OneMine
This is my Resource pack based off the game, [Oneshot](https://store.steampowered.com/app/420530/OneShot/)!

This resource pack make the UI similar to the UI from OneShot. Check the gallery to see what it looks like!

This resource pack also adds some small things from OneShot:

* Sheep have new sounds
* New Title Screen splashes!
* New Sound effects

Depends on my free time. If you have a mod you want this to support, check the GitHub and send an issue, or send a comment (CurseForge and Planet Minecraft only)
